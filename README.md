# nvim-config
Config for nvim in Windows10 using vim-plug (you need to install it)

## Pre-requisites:
- Git (to install plugins to your neovim)
- Chocolatey (to install neovim)
- NeoVim (the editor)

### Extra Pre-requisite:
- RipGrep

## Installation
- Run command `choco install neovim` and `choco install ripgrep`
- Then, go to `C:\Users\YourUser\AppData\Local` and create a folder called `nvim`
- Then, You will need [vim-plug](https://github.com/junegunn/vim-plug) and get the [plug.vim](https://github.com/junegunn/vim-plug/blob/master/plug.vim) (or the autoload folder in the repo) and copy the file into your nvim directory in a folder called `autoload`
- In your nvim folder (not autoload) copy the file called [init.vim](https://github.com/HankB-o-t/nvim-config/blob/main/init.vim) and paste in your nvim folder

#### And That's it!
